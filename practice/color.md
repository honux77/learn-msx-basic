# MSX Basic COLOR

## COLOR

- 색깔을 바꾸는 명령

## Syntax

```
COLOR <Foreground>,<Background>,<Border>
```

- `,`를 사용해서 각 매개변수 생략 가능

## Example

```
COLOR 1, 13, 5
COLOR 10
COLOR ,,13
```

## 주요 정보

### 스크린에 따른 색깔 종류

| SCREEN          | COLOR |
|-----------------|-------|
| 0-5             | 0-15  |
| 6               | 0-31  |
| 7               | 0-15  |
| 8               | 0-255 |
|                 |       |
